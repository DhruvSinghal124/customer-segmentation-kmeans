# Customer Segmentation using K-Means

This project performs customer segmentation using K-Means clustering based on income and spending score.

## Features
- Data preprocessing
- Feature scaling
- Elbow Method
- Silhouette Score
- Data visualization

## Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## How it works
- Load the dataset
- Select income and spending features
- Apply K-Means clustering
- Visualize customer segments

## Output
The model groups customers into different clusters based on income and spending behavior.
